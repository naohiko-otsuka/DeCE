ENDFIO Functions

These functions, given in endfio.cpp, read (ENDFReadMF) an MF/MT data section in an ENDF-6 file into an ENDF object, or print (ENDFWriteMF) the content of the ENDF object in a proper format depending on the section MF/MT number. Note that MF=7 has not been implemented yet.

ENDFRead

int ENDFRead (ifstream *fp, ENDF *lib, const int mf, const int mt)

Parameters
    *fp
        pointer to input stream
    *lib
        pointer to ENDF object
    mf, mt
        MF/MT numbers 
Return Value
    current data block index

Copy the MF/MT section in an opened ENDF file into an ENDF object. The data structure stored in the ENDF object

ENDFPrintLIST, ENDFPrint1Dim

void ENDFPrintLIST (ENDF *lib, const int idx)
void ENDFPrint1Dim (ENDF *lib, const int idx)

Parameters
    *lib
        pointer to ENDF object
    idx
        index to the data block to be printed
Return Value
    none

These two functions print an idx data block stored in the given ENDF object. ENDFPrintLIST prints the floating point data array in one column. ENDFPrint1Dim expects the ENDF object is for the TAB1 record, and print the (x,y) pairs.
