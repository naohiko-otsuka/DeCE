Header Manipulation

showheaders / showheadertext (print parameters in header)

The first 4 lines in a ENDF file contain information on identification, version, and some general properties of the file. Additionally, the next 5 lines may contain some more information, although these lines are not strictly required by the format. The showheaders command prints these values and text data. 

showheaders

header: ZA     2.705900e+04
header: AWR    5.842690e+01
header: AWI    1.000000e+00
header: NLIB              0  library identifier
header: LREL              0  release number
header: NVER              8  version number
header: NSUB             10  sublibrary number
header: NMOD              0  modification number
header: LDRV              0  0: primary evaluation, 1: derived
header: NFOR              6  library format
header: LRP               1  -1: no MF2, 0: no resonances, 1: resolved and/or unresolved, 2: PENDF
header: LFI               0  0: no fission, 1: fission
header: EMAX   2.000000e+07  upper limit energy
header: TEMP   0.000000e+00  temperature
header: ELIS   0.000000e+00  excitation energy of target
header: STA    0.000000e+00  0: stable, 1: unstable
header: LIS               0  state number of target
header: LISO              0  isomeric state number
[0---+----1----+----2----+----3----+----4----+----5----+----6----+-]
[ 27-Co- 59 LANL,ORNL  EVAL-JAN16 T.Kawano,G.Desaussure+           ]
[                      DIST-                                       ]
[----ENDF/B-VIII.0     MATERIAL 2725                               ]
[-----INCIDENT NEUTRON DATA                                        ]
[------ENDF-6 FORMAT                                               ]

The text data in the last 5 lines can be extracted by the showheadertext command. This command works only if the header section contains these lines, namely the header is given in a standard form.

showheadertext

  ZSYMAM: 27-Co- 59
    ALAB:LANL,ORNL
    AUTH:T.Kawano,G.Desaussure+
   REFER:
   EDATE:EVAL-JAN16
   DDATE:DIST-
   RDATE:
  ENDATE:
 LIBNAME:ENDF/B-VIII.0
  SUBLIB:INCIDENT NEUTRON DATA
  FORMAT:ENDF-6 FORMAT

editheader (modify header parameter)

To change an individual parameter in the header section, use editheader followed by the parameter name and value.

   editheader  ParameterName  Value

The parameter names are NLIB, LREL, NVER, NSUB, NMOD, LDRV, NFOR, LRP, LFI, EMAX, TEMP, ELIS, STA, LIS, and LISO. See ENDF-6 manual for the meaning. ZA, AWR, and AWI are not allowed to change. When the version number NVER is set to 10,

   editheader NVER 10

editheadertext (modify header text field)

Similar to editheader, the text filed data in the 5 optional lines can be modified by the editheadertext command. Again, this command works only if the header section is in the standard form.

   editheadertext  TextName  "Value"

where the allowed text names are ZSMYAM, ALAB, AUTH, REFER, EDATE, DDATE, RDATE, ENDATE, LIBNAME, SUBLIB, and FORMAT. The value must be given in the double-quotations. When a given text is longer than the defined field length, the extra letters will be cut. When the given text is shorter, blank padding will be made.

To change the laboratory name, 

   editheadertext ALAB "LANL"
