Options (set / unset)

set (setting global options)

You can control some DeCE behavior by setting option. For example, as a default, DeCE does not print line numbers, as it is obsolete. However, you can still activate the line number option by setting 

   set LineNumber

while to de-activate,

   unset LineNumber

Currently there are 7 options implemented in DeCE. Besides LineNumber above, we have 6 options that require an optional value, such as 

   set AngleStep 10
   set Output "externalfile.dat"

here the angle increment is set to 10 degree when printing the scattering angular distribution.

AngleStep
angle inrement when scattering angular distribution is printed
ReadXdataConversion
energy unit conversion when cross section data are imported
ReadYdataConversion
cross section unit conversion
WriteXdataConversion
energy unit conversion when cross section data are tabulated
WriteYdataConversion
cross section unit for tabulation
Output
file name to which the output will be redirected

read and multiread commands import data from an external file. As a default, DeCE assumes that the cross section is in mill-barn, and the energy is in MeV. When ReadXdataConversion is set to 0.001, the energy in the external file will be in keV.

WriteXdataConversion and WriteYdataConversion work only for the table command applied to MF3. The printed energies and cross sections will be multiplied by these factors. For example, when WriteYdataConversion is 1000, the output cross sections are in milli-barns, intead of the default of barns.

Setting Output changes the DeCE output from screen to a file. The file name is given as an option of this command (must be double-quoted.) When the file exists, the output will be appended. This is effective for the command table and extract. When unset Output , DeCE prints data on the screen.

Memory Usage

memoryusage (print memory usage table)

   memoryusage

This command prints a table of used computer memory for each subsection. Allocated is the size of allocated memoy in byte, and Used is the memory size actually data are stored.

  MF  MT   Record                  Integer                 Double                  Total
           Allocated        Used   Allocated        Used   Allocated        Used   Allocated        Used
--------------------------------------------------------------------------------------------------------
   2 151         640         480          40           0       25600       13056       26280       13536
   3   1         320          64          40           8       25600       19008       25960       19080

...

  33 600         320          96          40           0         800          48        1160         144
  33 800         320          96          40           0         800          24        1160         120
--------------------------------------------------------------------------------------------------------
   Total                                                                             8017520     6352624

For example, MF/MT=3/1 may be in the TAB1 data, where two integral numbers represent a set of interpolation ranges. Since the integer buffer contains these two numbers, the used memory is 4 x 2 = 8 bytes.
