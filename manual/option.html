Options (set / unset)

set (setting global options)

You can control some DeCE behavior by setting option. For example, as a default, DeCE does not print line numbers, as it is obsolete. However, you can still activate the line number option by setting 

   set LineNumber

while to de-activate,

   unset LineNumber

Currently there are 6 options implemented in DeCE. Besides LineNumber above, we have 5 options that require an optional value, such as 

   set AngleStep 10

here the angle increment is set to 10 degree when printing the scattering angular distribution.

AngleStep
angle inrement when scattering angular distribution is printed
ReadXdataConversion
energy unit conversion when cross section data are imported
ReadYdataConversion
cross section unit conversion
WriteXdataConversion
energy unit conversion when cross section data are tabulated
WriteYdataConversion
cross section unit for tabulation

read and multiread commands import data from an external file. As a default, DeCE assumes that the cross section is in mill-barn, and the energy is in MeV. When ReadXdataConversion is set to 0.001, the energy in the external file will be in keV.

WriteXdataConversion and WriteYdataConversion work only for the table command applied to MF3. The printed energies and cross sections will be multiplied by these factors. For example, when WriteYdataConversion is 1000, the output cross sections are in milli-barns, intead of the default of barns.
