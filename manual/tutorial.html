Quick Tutorial

Edit Comment Section

One of the most cumbersome things for editing an ENDF-6 format file is to modify the comment section (MF=1, MT=451). DeCE was designed to perform this very easily. Suppose we added some lines in the comment section like this:

 3.307500+4 7.427800+1          1          0          0          03325 1451    1
 0.000000+0 0.000000+0          0          0          0          63325 1451    2
 1.000000+0 2.000000+7          0          0         10          73325 1451    3
 0.000000+0 0.000000+0          0          0        299        2803325 1451    4
 33-As- 75 LANL       EVAL-MAR10 T.Kawano                         3325 1451    5
                      DIST-MAR10                                  3325 1451    6
----ENDF/B-VII        MATERIAL 3325                               3325 1451    7
-----INCIDENT NEUTRON DATA                                        3325 1451    8
------ENDF-6 FORMAT                                               3325 1451    9
                                                                  3325 1451   10
****************

This is a test file

================================================================= 3325 1451   11
Evaluation Summary                                                3325 1451   12
                                        T. Kawano (LANL) Mar 2010 3325 1451   13
                                                                  3325 1451   14
Cross sections updated based on CoH3[Ka10] calculations.          3325 1451   15

*************************** DICTIONARY ***************************3325 1451  303
                                1        451        583          03325 1451  304

Between the line number 10 and 11, new 4 new lines are added. There is no MAT/MF/MT numbers in these new lines. The total number of the comment section, which was 299 given at the forth line, but now should be 304, because we added 4 lines. In addition, the total number of lines for the MF=451 section in the dictionary must be fixed too. DeCE modifies these numbers automatically.

When we are editing a file ENDF.DAT, a new file ENDF.NEW can be created with the DeCE code as following.

  % dece -o ENDF.NEW ENDF.DAT
  quit

Since DeCE will be in the interactive mode when invoked, you need to type quit to start creating the new file. Instead of typing quit, DeCE accepts the -q option, which tells DeCE to quit immediately.

  % dece -q -o ENDF.NEW ENDF.DAT

In the new file ENDF.NEW, the number of total lines for the comment section is properly fixed. Note that DeCE(Pyrite and later) no longer prints the line numbers. You can print the line number by setting the LineNumber option. See the option section.

Tabulate Data

DeCE can convert ENDF-6 formatted data into a simple tabulated format. This can be done with the command line options -f MF and -t MT, where MF and MT are the MF/MT numbers to be processed.

For example, to extract the (n,2n) cross section, MF is 3 (file number), and MT is 16 (reaction number).

 % dece -f3 -t16 ENDF.DAT
#    [ 3325 :  3 :  16 ]    33 -  75
# Cross section
#           QM -1.024325e+07  mass difference Q-value
#           QI -1.024325e+07  reaction Q-value
#           NR             1  number of interpolation range
#           NP            17  lin-lin  interpolation
  1.038116e+07  0.000000e+00
  1.050000e+07  4.433555e-04
  1.100000e+07  5.322542e-02
  1.150000e+07  2.434272e-01
  1.200000e+07  4.474358e-01
  1.250000e+07  6.361240e-01
  ...           ...

In this case, DeCE won't be in the interactive mode, but will exit rignt after the data are tabulated.

I have not yet completed the data tabulation function in DeCE. Some sections, such as thermal scattering, cannot be processed for now.

Interactive Editing

In the interactive mode, DeCE accepts many commands to manipulate numerical data in the file. Often the command is followed by the MF/MT numbers to specify which section will be targeted.

As an easiest example, the extract command copies one section from the file and print it as is. First, just start DeCE with the ENDF-6 formatted file, then type this command with the section MF/MT numbers.

 % dece ENDF.DAT
extract 12 51
 3.307500+4 7.427800+1          2          1          2          0332512 51
 1.986060+5 0.000000+0          0          0          2          1332512 51
 0.000000+0 1.000000+0                                            332512 51
                                                                  332512  0
quit

Since DeCE is still in the interactive mode, type quit to stop the program.

When your operation is more complicated, it is better to put all the commands in a file, then run DeCE by giving these commands from the standard input as a usual UNIX manner. The next example creates a new section MF/MT=3/5 as the sum of some other partial cross sections.

First we prepare a new file input.dat, containing the following lines.

calc 5 = 1 + 2
calc 5 = 5 + 4
calc 5 = 5 + 16
calc 5 = 5 + 102
extract 3 5
quit

The first line means, MF/MT=3/5 is first constructed as the sum of total (MT = 1) and elastic (MT = 2) cross sections. Then the other sections, MT = 4 (inelastic), 16 (2n reaction), and 102 (capture) are added to MT = 5. The calc command works on MF3 only.

We feed the input file to DeCE.

 % dece ENDF.DAT < input.dat
 3.307500+4 7.427800+1          0          0          0          03325 3  5
 7.329602+6 7.329602+6          0          0          1         953325 3  5
         95          2                                            3325 3  5
 1.000000-5 0.000000+0 2.530000-2 0.000000+0 1.000000+5 0.000000+03325 3  5
 1.000000+5 1.748307+1 1.500000+5 1.604954+1 2.000000+5 1.512379+13325 3  5
 2.012798+5 1.510618+1 2.500000+5 1.443547+1 2.682211+5 1.422633+13325 3  5
....

When -o option is given, DeCE writes the new section to the file, together with all other sections. This is the most convenient to re-adjust the elastic scattering cross sections (MT = 2), by subtracting all the partial cross sections from the total cross section. This instruction can be as following.

make4
calc 3 = 4 + 16
calc 3 = 3 + 17
calc 3 = 3 + 22
calc 3 = 3 + 28
calc 3 = 3 + 102
calc 3 = 3 + 103
calc 3 = 3 + 107
calc 2 = 1 - 3
calc 1 = 2 + 3
delete 3 3
quit

 % dece ENDF.DAT -o ENDF.NEW < input.dat

The make4 command is an abbreviation for adding subsections from MT = 51 to 91. Firstly the non-elastic cross sections are calculated by adding all the partial cross sections (this depends on which sections are given in the input ENDF-6 file), then the elastic cross sections are calculated by subtracting the non-elastic from the total (calc 2 = 1-3). Finally the total cross section is again reconstructed by adding the elastic and non-elastic cross sections. This final step ensures that the section for total cross section (MT = 1) contains all the energy points appeared in the entire file.
