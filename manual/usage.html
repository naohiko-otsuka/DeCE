How to Run the Code

Command-Line Options

DeCE runs in three different modes depending on how command-line options are provided.

The first one is like below: an ENDF-6 data file is only provided without any command-line options.

 % dece  ENDF_in.DAT

DeCE will be in an interactive mode, and prints a manipulation result as indicated by a user. DeCE continues running until you type quit, end or exit.

The second one is to use an option of -o File_Name.

 % dece  -o ENDF_out.DAT ENDF_in.DAT

Again DeCE will be in the interactive mode. Once DeCE processed all the instructions provided, and read quit command, the code writes the whole ENDF-6 formatted data on the ENDF_out.DAT file.

DeCE can be used just only for renumbering, or for reformatting the comment section, without any numerical manipulation. In this case, you can bypass the quit command by a command line option -q, which tells DeCE to exit right away and reformat the file. 

 % dece  -q -o ENDF_out.DAT ENDF_in.DAT

The last mode is to tabulate a specific section in the ENDF file.

 % dece -f  MF  -t  MT   ENDF_in.DAT

These options -f and -t tells DeCE to extract that MF/MT section, and print the data in a human readable format. MF2, the resonance parameters, is a special case. Since MT is always 151, you can omit the -t option. Note that this function has not been completed yet. Some MF/MT sections cannot be processed.

and when all the input commands are in a file,

 % dece  ENDF_in.DAT  <  input.dat

More Options

There are some more command-line options, DeCE recognizes:

-r
Print thermal cross sections calculated from resonance parameters.
-e En
Print a cross section at a given energy En in MF3. MT must be specified by -f 3 -t MT .
-v
Verbose mode with annoying output.
-h
Print help, and quit.
-s
Print a complete list of MF/MT numbers in the file, and quit.
